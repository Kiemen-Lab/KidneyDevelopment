# KidneyDevelopment
Overview

This repository contains the code used to generate the analyses and figures presented in the manuscript:

"Cross-species Whole-organ Reconstruction Reveals Conserved Cellular Programs and Divergent Architecture During Kidney Development."

The repository is organized into folders corresponding to the figures in the manuscript (Figure1–Figure5) and includes a DeepLearning folder containing the scripts used for tissue segmentation and model inference.

System requirements
Software
MATLAB R2023b (tested)
Python 3.10 or later
Operating systems

The code has been tested on:

Windows 10
Windows 11

Most analyses can be run on a standard desktop computer (≥16 GB RAM recommended). An NVIDIA GPU with CUDA support is recommended, but not required, for deep learning inference.

Instructions for use

Each figure folder contains the scripts required to reproduce the analyses shown in the corresponding figure. To analyze your own data, update the input and output paths in the scripts and execute the pipeline as described in the folder documentation.

Data availability

The datasets supporting this study will be made publicly available upon publication of the manuscript.
